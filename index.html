    Import Library

    YouTube Music Import using Google Takeout
    How to get your data
    1. Go to takeout.google.com
    2. Click "Deselect all", then select only "YouTube and YouTube Music"
    3. Click "All YouTube data included" and select: history, music-library-songs, playlists
    4. Export as .zip, download and extract
    Spotify Import playlists using Exportify
    How to export from Spotify
    1. Go to exportify.net (free third-party tool)
    2. Log in with your Spotify account
    3. Click "Export" next to each playlist, or "Export All"
    4. CSV files are downloaded with columns: Track Name, Artist Name(s), Album Name
    5. Select the downloaded CSV files below

    💡 Each CSV file becomes a separate playlist

    Apple Music Import from Apple Music Library export
    How to export from Apple Music
    1. Open the Music app on your Mac
    2. Select a playlist in the sidebar
    3. Go to File → Library → Export Playlist...
    4. Choose Plain Text or XML format
    5. For full library: File → Library → Export Library...

    💡 Plain Text exports as tab-separated values (TSV). XML exports your entire library.
